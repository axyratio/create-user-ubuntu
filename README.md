# create-user-ubuntu
## install wsl ubuntu

## open your ubuntu
```
ubuntu
```

before install your os, please check your os.

```
wsl --install
```

then restart.

## add user
```
adduser user
usermod -a -G sudo user
su - user
```

## del user
```
deluser user
```
